About

EnrollMe is a scripting language with the purpose of providing a simple framework to make the enrollment process easier to use. Students are able to select and modify their schedule using simple commands.

Motivation

Our motivation is to facilitate the currently confusing enrollment process. Many students, specially freshman, do not understand this process and usually seek help somewhere else. The goal is to provide students with a tool that is easy to use with clear commands and simple vocabulary.

Language Features

EnrollMe provides basic features for the enrollment process:

  • Perform common tasks related to class enrollment like: enroll a class, drop a class and changing a class section.
  • Show the student schedule
  • Display available courses using filters like courseID, professor name and/or time.
